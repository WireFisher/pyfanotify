``pyfanotify`` - is a Python wrapper for the Linux ``fanotify``.

.. seealso::
   `man fanotify <https://man7.org/linux/man-pages/man7/fanotify.7.html>`_ for more info on ``fanotify``.

.. note::
   Required ``python 3.6+``

.. warning::
    Required execution from **ROOT**
